# Sample Videos

Do not commit the event sample videos to this repo.

For demos, keep the sample video bundle in one of these places:

- On the Spark: `/home/nvidia/Videos/dev-profile-sample-data`
- On the demo laptop: any local folder the browser can upload from
- In an external download location, such as Google Drive, SharePoint, NGC, or a GitHub Release asset

The VSS browser UI uploads from the computer running the browser. If the UI is opened from a laptop, the videos need to be on that laptop. The Spark-local copy is still useful for backup, API upload workflows, and repeatable event setup.

This directory is intentionally ignored by Git except for this README. If an operator wants repo-relative paths on an event machine, they can copy or unzip the sample bundle here locally.
